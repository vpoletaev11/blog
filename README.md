# Blog service

## Download porject
```shell
$ git clone https://github.com/vpoletaev11/blog
```
**[!]** Now change directory to downloaded project in order to perform instructions listed below
```shell
$ cd blog
```

## Test project
```shell
$ go test ./...
```

## Run project with docker-compose
```shell
$ sudo docker-compose up
```

## Run project locally
Build project:
```shell
$ go build -o app .
```
Run project:
```shell
$ ./app
```
